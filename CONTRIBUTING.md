\# Quy tắc đóng góp — Dự án TaskMaster Docs



\## Commit Message

Format: `\[Tên] Hành động: Mô tả`

\- Hành động: Thêm, Sửa, Xóa, Định dạng, Sửa lỗi



\## Branch

Format: `feature/tên-người/tên-phần-việc`

\- Dùng chữ thường, không dấu, dấu gạch ngang



\## Pull Request

\- Tiêu đề rõ ràng

\- Mô tả đầy đủ các thay đổi

\- Gán ít nhất 2 reviewer



\## Review

\- Đọc kỹ toàn bộ nội dung

\- Comment xây dựng, cụ thể

\- Approve hoặc Request Changes kèm giải thích



\## Quy trình

1\. Pull main mới nhất

2\. Tạo branch riêng

3\. Viết nội dung + commit thường xuyên

4\. Tạo PR + gán reviewer

5\. Sửa theo feedback

6\. Merge sau khi được approve
